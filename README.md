# Student Retention & Performance Prediction System

## Project Overview

The Student Retention & Performance Prediction System is a Machine Learning and Power BI project developed to analyze student academic performance and identify students who are at risk of poor outcomes. The project integrates assessment scores, engagement metrics, demographic information, registration records, and Virtual Learning Environment (VLE) activity to generate meaningful insights for educators and administrators.

By combining predictive analytics with interactive dashboards, the system helps educational institutions monitor student performance, improve engagement, and support timely academic interventions.

---

## Project Objectives

- Predict student academic performance using Machine Learning.
- Identify students at risk of poor academic performance.
- Analyze student engagement and learning behavior.
- Build interactive Power BI dashboards for performance monitoring.
- Support data-driven decision-making for academic institutions.
